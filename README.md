# ComCody Lab v68 — Build · Play · Share

ComCody turns game logic into visible, reusable pieces.

v68 adds the first no-code creator bridge: **save the current Program Stack as a new reusable block**.

## What users can do

- Play Maze or imported puzzle games.
- Remix the Program Stack.
- Save the working stack as a new block.
- Add that block back from the Piece Store.
- Export the block or full game as a ZIP.

## Creator levels

1. **Player:** play and remix.
2. **No-code creator:** save visible pieces as a new block.
3. **Coder:** inspect YAML, JS, tests, and bundle metadata behind Expert.

## v68 focus

- No-code Piece Creator in Packs.
- Story, tags, license, and dependency chain travel with the created block.
- Front UX stays clean; creation tools stay behind Packs.


## v68 — Tablet Fit-to-Content UX

Builder panels now size to useful content instead of stretching into empty space. Tile tools stay compact, Actions stay visible, and Quests render compactly for tablet users. No game/runtime logic changed from v65.


## v68 Tablet Builder Fit

Builder now uses gamer-facing **Maze Tiles** instead of the old builder wording. Fit mode shows only core setup tiles, avoids the visible inner scrollbar, and All reveals advanced/debug tiles only on demand.
