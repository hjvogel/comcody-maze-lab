# ComCody Tetris Lite Pack

Import this ZIP in **ComCody Lab → Tetris Builder**.

It adds a small set of Tetris modules:

- Board Grid
- Tetromino Tray
- Falling Piece
- Move Left / Right
- Rotate
- Soft Drop
- Lock Piece
- Clear Lines

Use **Auto Assemble** or drag the modules into the Tetris Assembly Stack. When all required modules are present, launch the imported Tetris runtime.
